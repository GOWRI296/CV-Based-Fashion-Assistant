# 👕 Outfit Color Matcher (Computer Vision Project)

## 🚀 Overview

This project detects the color of a user's outfit in real-time using a webcam and suggests matching colors for styling. It uses Computer Vision techniques like image processing, color detection, and filtering.

---

## 🧠 Features

* Real-time webcam input
* Dominant color detection
* Skin filtering for better accuracy
* Outfit matching suggestions
* Visual display with color boxes

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy

---

## 📦 Installation

```bash
pip install opencv-python numpy
```

---

## ▶️ How to Run

```bash
python outfit_color_matcher.py
```

(Optional webcam test)

```bash
python camera_test.py
```

---

## 📸 Usage

1. Run the program
2. Place your shirt inside the green box
3. The system detects the color
4. Matching outfit suggestions are displayed

---

## 📁 Files

* outfit_color_matcher.py → Main project
* camera_test.py → Webcam testing
* test_run.py → Basic test script
